# Análise do Descompasso Institucional e Subnotificação na Rede de Proteção à Mulher no Rio Grande do Sul

Este repositório apresenta um projeto de engenharia e análise de dados voltado à auditoria da rede de proteção à mulher no Estado do Rio Grande do Sul. O trabalho cruza dados de Segurança Pública (SSP-RS) e Saúde Pública (DATASUS) para identificar falhas de integração e subnotificações municipais.

## Estrutura do Repositório

Para facilitar a avaliação da banca examinadora, os arquivos foram organizados da seguinte forma:

*   **`notebooks/`**: Contém o arquivo `.ipynb` com todo o código Python desenvolvido para a Limpeza, Tratamento e Análise Exploratória de Dados (EDA).
*   **`Analise_Subnotificacao_Violencia_RS.pdf`**: Relatório técnico oficial contendo o detalhamento completo das etapas de **Coleta de Dados**, **Modelagem** e **Conclusões/Diagnósticos**.
